RÔDEUR

Equipez-vous d'un Rôdeur. Tirez pour envoyer le Rôdeur sur une ligne droite. Le Rôdeur se verrouille sur les ennemis ou les pistes qui entrent dans son cône de vision frontal et les suit ; s'il atteint un ennemi, il le rend myope. Maintenez la touche de tir pour guider le Rôdeur dans la direction de votre réticule.
